# Hola Mundo con Typescript

Plantilla inicial para Vite + TypeScript con Tailwind CSS v4 ya configurado.

## Ejecutar en local

```bash
npm install
npm run start
```

## Scripts disponibles

- `npm run start`: Inicia el servidor de desarrollo de Vite.
- `npm run dev`: Alias de start.
- `npm run build`: Hace type-check y build de produccion.
- `npm run preview`: Previsualiza el build de produccion.

## Notas

- Esta plantilla incluye un asset de Hello Rigo para verificar archivos estaticos y estilos.
- Si puedes ver la tarjeta de Rigo y el mensaje, Tailwind esta funcionando.
